# Changelog

All notable changes to AnswerLens will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-04

### Added
- Initial release of AnswerLens
- Screen capture functionality (fullscreen, window, region, fixed region)
- Google Gemini AI integration for image analysis
- Professional teleprompter with auto-scroll feature
- Live preview of captured screens
- Continuous monitoring mode (1-minute intervals)
- API key configuration and storage
- Screenshot export functionality
- Reading line marker in teleprompter
- Adjustable font sizes and scroll speeds
- Cross-platform support (Windows, Linux, macOS)
- Logo icon support for application window
- Comprehensive documentation

### Features
- Multiple capture modes for flexible screen analysis
- Free Google Gemini API integration
- Teleprompter with customizable display settings
- Auto-start scrolling with configurable delay
- Show/hide control panel for distraction-free reading
- Keyboard shortcuts (Space, Escape)
- Semi-transparent overlay mode
- Always-on-top teleprompter window

### Technical
- Python 3.8+ compatibility
- Tkinter-based GUI
- MSS library for screen capture
- Pillow for image processing
- Google GenAI SDK integration
- pywin32 for Windows-specific features

## [Unreleased]
